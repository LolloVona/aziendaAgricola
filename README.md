# 🌱 Applicazione Web per Azienda Agricola

## 📌 Descrizione del progetto

Questo progetto consiste nello sviluppo di un'applicazione web dedicata alla gestione e alla promozione di un'azienda agricola.

L'obiettivo principale del team si focalizza su questi quattro macropunti:

- Sia logicamente corretta senza bug
- Sia utilizzabile da ogni utente (esperto-inesperto)
- Design Eccellente
- Sicuro 


L'applicazione sarà sviluppata utilizzando tecnologie web (frontend e backend) con particolare attenzione a usabilità, design e performance.
